# MerchTrax

MerchTrax is a mobile application designed to help manage store visits and track history efficiently.

## Requirements
  If created using the Tool/Platform recommended in the Final Project:

    Fun or useful 10%
    At least 3 Activities/Pages 30%
    A working list 30%
    A menu for navigation 10%  
    Look and Feel is Customized 10%
    A local Database or Use an API (Marvel, Google Maps, etc...) 10%
    Include something that was not covered in class (Push Notifications, Web Browser, Accelerometer, etc...) (Extra Credit)
  
  Extra Credit:
    Accessibility
    Localization
    Globalization
    ReAct Mobile
