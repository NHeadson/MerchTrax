# MerchTrax

MerchTrax is a mobile application designed to help manage store visits and track history efficiently.

## Rubric
In general, you are to create an App of your choice that meets the following requirements:

If created using the Tool/Platform recommended in the Final Project:

Fun or useful 10%
At least 3 Activities/Pages 30%
A working list 30%
A menu for navigation 10%  
Look and Feel is Customized 10%
A local Database or Use an API (Marvel, Google Maps, etc...) 10%
Include something that was not covered in class (Push Notifications, Web Browser, Accelerometer, etc...) (Extra Credit)
Extra Credit:

Accessibility
Localization
Globalization
ReAct Mobile

## Paper Requirements
This project satisfies the state requirement that students that graduate with a Web Software Developer degree must demonstrate the ability to Analyze New Technologies.

The requirement is referred to as a Program Outcome. The Outcome has three Criteria: 

research new technologies
implement new technologies
evaluate new technologies
The project has two deliverables: A paper and an App that must be demonstrated.

For the Paper, you will need to create a scoring scheme to evaluate a tool to create cross-platform mobile apps. You will need to research three tools (one is MAUI) to create cross-platform apps. 

The paper needs to include an overview of the research and a recommendation for the tool of choice. Additionally, the paper must include the scoring guide filled out.

The paper must be submitted as a PDF it should be more than two pages but not over six. Your scoring should also be included as an appendix.

The App is to be made using the tool that you selected. The App does not have to be complicated. It should include input and output from the screen and multiple screens/pages and should have a custom look & feel.

Rubric for Paper:

Platform Scoring Guide 30% - The guide is well thought out, filled out, and reasonable.

Research 30% - Three tools/platforms are researched
Sources are cited
At least three sources are used.

Paper 30% - The paper has the correct sections and is sufficient quantity and quality.
