# MerchTrax

MerchTrax is a mobile application designed to help manage store visits and track history efficiently.

## Features
- Home screen with a counter and navigation.
- History screen displaying past visits.

## Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Configure Firebase by replacing `firebase.js` with your Firebase project details.
4. Start the app with `npm start`.

## Build
To build the app for Android:
```
eas build --profile production --platform android
```

For more information, visit the [Expo documentation](https://docs.expo.dev/).
